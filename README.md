## Install

```shell
git clone https://github.com/korbsstudio/eOS-Dark-Plank-Theme
cd eOS-Dark-Plank-Theme
./install.sh
```


## Enable Theme
Hold ctrl and right click anywhere on Plank and select settings.
Then change theme to eOS-Dark.
